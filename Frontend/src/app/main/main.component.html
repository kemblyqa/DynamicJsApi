¿Quiénes somos?
Coffy es un API abierto para la generación de manera dinámica de código Javascript, permitiendo a nuestros usuario poder incorporar funciones desarrolladas por él o por la comunidad a sus proyectos.
¿Qué hacemos?
La plataforma ofrece la posibilidad de revisar un largo y cargado repositorio de funciones que están totalmente a la entera disposición de quién guste hacer uso de ellas. Estas funciones han sido desarrolladas por los mismos miembros de la comunidad de usuarios. En este portal, encontrá las herramientas para consultar o una vez registrado crear y generar sus propias funciones.
¿Qué Ofrecemos?
Acá podras encontrar la posibilidad de importar diferentes funciones utilizando tres diferentes métodos:
  • Incrustado en un etiqueta script.
  • Importación directa de módulos o funciones mediante un listado de códigos de las funciones que se quiere agregar y sus dependencias.
  • Importación basada en referencia URL y código de a función.
Además, la posibildad de crear un repositorio extenso de funciones o consultar aquellas desarrolladas por los usuarios de la comunidad.
Instructions
Importar Librería

Se puede incrustar el archivo JS utilizando etiqueta SCRIPT para cargar el archivo quien contiene las funciones básicas de importación

Importar Modulos o Funciones

La librería importada contiene un método denominado importar(), que recibe el listado de códigos de funciones a importar y descarga de forma dinámica las funciones solicitadas junto con sus dependencias.

Importación basada en referencia URL y código de función

El usuario exporta del sitio administrativo la referencia (URL) a la función requerida y la importa a su desarrollo mediante una sola etiqueta SCRIPT.
